## Installations

* headlessui
============= 
- npm install @headlessui/react
- npm install @heroicons/react

* ShadCn
=============
- npx shadcn@latest init
- npm shadcn@latest add component_name

* toast (to show notifications)
================================

- npm i sonner
- npx shadcn@latest add sonner
- npm i react-toastify 
`Instructions:`
(INSIDE LAYOUT- import ToastContainer from react-toastify and import react-toastify/dist/reactToastify.css)-place the container anywhere in the layout

* react font awesome elements
==============================
- npm i react-icons
Import chevron for back redirect or FaUser, FaSignInAlt, FaSignOutAlt, FaBuilding from react-icons/fa

* axios
============
- npm i axios

* GSAP useGSAP
==============
- npm i gsap @gsap/react

* TanStack/react-query
=======================

npm i @tanstack/react-query
